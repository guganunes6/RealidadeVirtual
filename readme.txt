--- INSTRUCTIONS ---

1. Clone repository
2. Open project with Unity 2019.4.13f1 (be sure to have both Android Build Support and Universal Windows Platform Build Support Installed)
3. Choose testing scene in Assets > Scenes > FinalScenes:
	Keyboard & Mouse -> KBM
	Keyboard & Mouse + VR -> Mix_VR_KBM
	VR -> VR
4. Press play
4.1 Keep in mind that to run Mix_VR_KBM and VR scenes you will need to have the Oculus Quest (1 or 2) or Oculus Rift connected to the PC, as well as the Oculus application running on the background and Android Studio installed
